- Use of flexbox for positioning
```css

  display: flex;
  align-items: center;
  gap: 15px;
}

```
- Fitting images as background
```css
body{
  background-color: var(--pale-blue);
  background-image: url(../images/pattern-background-mobile.svg);
  background-size: cover;
  background-repeat: no-repeat;
}
```
